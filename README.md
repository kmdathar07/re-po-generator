# Re-Po Generator
### Resume + Portfolio Builder

A full-stack SaaS app to build professional resumes and animated portfolios.

**Live App →** [re-po-generator.vercel.app](https://re-po-generator.vercel.app)

---

## Features
- 7 Resume Templates (Corporate, Modern, Tech, Startup, Creative, Academic, Minimal ATS)
- PDF Download powered by Playwright
- Animated Portfolio with 4 pages (Home, About, Projects, Resume)
- Supabase Auth + Cloud Save
- 100% Free to use

## Tech Stack
- **Frontend:** React + Vite + Tailwind CSS
- **Backend:** FastAPI + Playwright (PDF)
- **Database:** Supabase (PostgreSQL)
- **Hosting:** Vercel (frontend) + Render (backend)
